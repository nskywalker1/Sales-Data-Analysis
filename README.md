## ğŸ“ Project Overview
This project is an interactive Power BI dashboard designed to analyze retail sales performance for an electronics and clothing retailer in India.

The goal of this dashboard is to provide stakeholders with actionable insights into sales trends, product performance, and the effectiveness of marketing promotions. It enables data-driven decision-making to optimize inventory and marketing strategies.

**Key Business Questions Answered:**
* Which product categories drive the highest revenue?
* How do different promotion types (e.g., Summer Sale, Diwali) impact net sales?
* What are the top-performing regions (States/Cities)?
